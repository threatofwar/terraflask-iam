# terraflask
Terraform API Service
=======

git clone this repo
```bash
cd terraflask
```
```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```
```bash
pip install Flask python-dotenv
```
copy .env.example to .env
```bash
cp .env.example .env
```
edit the newly created .env file
```bash
python app.py
```
